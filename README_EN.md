## Features ⚙️

* Take a screenshot on the phone and transfer to the PC
* Process the image with Breadth First Search to find the solution
* Obtain swipe commands and relay them to the phone

## Getting Started 🚀

<details>
<summary>Click on me for expand to see guide for Windows (for beginner's)</summary>

- **Step 0.** Install Python and VSCode:

> Python: https://www.python.org/downloads <br>
> VSCode: https://code.visualstudio.com/download

- **Step 1.** Clone the repository:

```bash
git clone https://github.com/zdky/hamsterkombat-minigame-solver.git
```
> or click the button above [Code] ➜ [Download ZIP]

- **Step 2.** Open the project folder and type in the address bar:
```bash
cmd
```

- **Step 3.** Create a python virtual environment. At the «command prompt» that opens, enter the command:

```bash
python -m venv .
```

- **Step 4.** Right click in project folder and «Open with Code»

- **Step 5.** Select your virtual environment. In VSCode click «Ctrl+Shift+P» and type:
```
python select interpreter
```
«Enter» and click on your venv, like: «Python 3...('hamsterkombat-minigame-solver': venv)...»

- **Step 6.** Now open Terminal. VSCode menu: «Terminal» ➜ «New Terminal» or Ctrl+Shift+`

- **Step 7.** Install requirements, type in VSCode terminal:

```bash
pip install -r requirements.txt
```

- **Step 8.** Now setup your phone:<br>1) Go to «Settings» ➜ «About phone»<br>2) Scroll down to «Build number»<br>3) Tap «Build number» seven times<br>4) If you see message «You are now a developer» — good!<br>5) Go to «Settings» ➜ «System» or «Additional settings» ➜ «Developer options»<br>6) Find «USB debugging» turn ON<br>7) Find «USB debugging (Security settings)» turn ON*<br>*(*if you need authorization in MIUI or other, do it!*)
  
- **Step 9.** Connect your phone via USB with PC

- **Step 10.** Open VSCode «Explorer» (Ctrl+Shift+E) and click on «main.py»

- **Step 11.** Start program first time for test. Click on ▷ in the upper right corner

- **Step 12.** Open mini-game in HamsterKombat click start and start program again.

- **Step 99.** Enjoy and follow on [my telegram channel](https://t.me/Zhidky)

</details>
